- **[FAISS Wiki](https://github.com/facebookresearch/faiss/wiki)** - Comprehensive guides and tutorials
- **[FAISS API Documentation](https://faiss.ai/cpp_api/)** - C++ API reference

### Research Papers
- **[Billion-scale similarity search with GPUs](https://arxiv.org/abs/1702.08734)** - Original FAISS paper (2017)
- **[Product Quantization](https://hal.inria.fr/inria-00514462v2/document)** - PQ compression method
- **[HNSW](https://arxiv.org/abs/1603.09320)** - Hierarchical Navigable Small World graphs
